# Portfolio Website

Personal portfolio built with React and Vite.

I use this repo to keep my site updated with new projects, experience, courses, and skills.

## Tech

- React
- Vite
- CSS

## Main Files

- Content/data: `src/data/portfolioData.js`
- Page layout/sections: `src/App.jsx`
- Styling/themes: `src/App.css`

## Run Locally

```bash
npm install
npm run dev
```

## Other Useful Commands

```bash
npm run build
npm run lint
```
